Locator Practice

Locators are used in automation to identify and interact with elements on a web page. Common types include:

  • ID: id="elementId" — Fastest and most reliable if unique.
  • Name: name="elementName" — Useful for form fields.
  • Class Name: class="className" — For elements with specific classes.
  • Tag Name: <button>, <input> — For generic element types.
  • Link Text/Partial Link Text: For anchor tags by visible text.
  • CSS Selector: .class, #id, div > span — Very flexible, supports complex queries.
  • XPath: //div[@id='main'] — Most powerful, supports complex DOM traversal.
  • Data/Test Attributes: data-testid, data-qa — Best for automation, avoids UI changes.
  • ARIA Attributes: aria-label, role — For accessibility and automation.
